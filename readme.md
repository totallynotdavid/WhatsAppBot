## Prerrequisites:

1. ffmpeg

## Steps

1. Create an `.env` file with the following variables: 

   - `spotifyClientId`
   - `spotifyClientSecret`
   - `youtubeKey`

2. Run `npm i github:pedroslopez/whatsapp-web.js#fix-buttons-list` for buttons. Also delete some comments.
3. Run the following commands:
   
   - `npm install`
   - `npm run start`
